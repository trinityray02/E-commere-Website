# 🛍️ Simple E-Commerce Website

A clean and responsive front-end e-commerce website built with **HTML**, **CSS**, and **JavaScript**.  
It showcases products, allows filtering by category, and includes a working shopping cart and checkout page.

---

## ✨ Features
- 🏠 **Home Page** – Displays featured products and categories  
- 🛒 **Shopping Cart** – Add, remove, and view items  
- 💳 **Checkout Page** – Displays order summary and total  
- 🔍 **Product Filtering** – Browse by categories  
- 🌈 **Modern UI** – Clean layout with responsive design  
- 💬 **Animations** – Smooth and subtle transitions  

---

## 🧠 Technologies Used
- **HTML5** – Structure and content  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Dynamic interactions
